# phone-catalog-app

### Environments:
There are two environments in the project, one backend and one frontend.
- Client
- Server 

To work individually on each project you only need to open the folder of your choice with your favorite code editor.

### Requirements
- Docker / docker-compose tool
- NodeJS
- Yarn or NPM.

### How to launch the project?
You can launch it very easily with docker, you just need to have docker installed and run the following command:

in the root of this file you should run:
`docker-compose up`

Look the app in `http://127.0.0.1:8080`
