# Mobile phone catalog service

### The objective:
This is a simple backend created with express to serve data to the phone catalog frontend.

### First step: Install dependencies:
Before executing any of the commands below, you must install all the necessary dependencies for the correct use of the software.
To obtain them, you will need to enter one of the following commands (this is your preference).

``yarn install``

or

``npm install``


### How to run the service:

If you use yarn as a package manager, use the following command to install the dependencies and run the service, otherwise you can run it using the node command.

``yarn start``

or

``npm start``


### Testing:
The example shows a testing performed with Jest and Supertest, in this case we will only do a check of the root endpoint and the list of mobile phones.

To launch the tests you only need to enter the following command:

``yarn test``

or

``npm run test``
